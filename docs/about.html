About

About the Project

This project was made for a local insurance company. It is a proof-of-concept on how to explain commonly confused terms relating to insurance. In the example video, the term "Gap Insurance" is explained in a simple manner in order to adhere to different kinds of consumers that need more information on insurance.

About the Process

The script for the video was written by Min Wu, while the video was animated, voiced, and edited by Tyler Akam. Tyler began his work by making assets in Adobe Illustrator, then continued by animating them in separate compositions in Adobe After Effects, then finally editing it all together in Adobe Premiere Pro. The royalty-free background music was sourced from YouTube Audio Library. The splash page for the website was made by Min Wu, while this about page was made by Tyler Akam. This website is being published by GitHub through GitHub pages. It is available to see here.
